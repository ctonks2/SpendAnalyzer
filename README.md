# Spend Analyzer

A CLI prototype of the Spend Analyzer project.

Goal: ingest CSV/JSON receipts, normalize and store transactions in memory, generate reports, and connect to an LLM (Mistral) for Q&A.

Quickstart

1. Create a Python 3.10+ virtual environment and activate it.
2. Install requirements: pip install -r requirements.txt
3. Run the CLI prototype: python main.py

Status: Prototype scaffold for Chunk 1 (CLI) including data manager, LLM client, and report generator.
